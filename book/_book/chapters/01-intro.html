1.2 AI 编程革命

从”写代码”到”引导代码”

2024年，编程的方式正在经历根本性变革。过去，程序员需要记忆大量语法规则，手写每一行代码。现在，AI 工具可以根据自然语言描述直接生成代码。

重要范式转变

传统编程教育的核心假设——“会写代码 = 会编程”——已经过时。在 AI 时代，理解问题、描述需求、验证结果才是核心能力。

传统编程 Vibe Coding
记忆语法规则 用自然语言描述需求
手写每行代码 AI 生成 + 人工审查
查阅文档找 API 直接问 AI
逐行调试找错误 让 AI 解释错误原因

Andrej Karpathy 的 Vibe Coding 哲学

Vibe Coding 这个概念来自 AI 领域先驱 Andrej Karpathy（OpenAI 创始成员、Tesla AI 前总监）：

“我称之为 Vibe Coding——你完全沉浸在氛围中，拥抱指数级发展，忘记代码的存在。”

核心观点：

  1. 描述问题，而非编写代码 —— 用自然语言告诉 AI 你想要什么
  2. 验证结果，而非死记语法 —— 关注代码是否正确运行
  3. 迭代改进，而非一次写对 —— 与 AI 对话，逐步完善
  4. 理解架构，而非背诵 API —— 把握整体设计，细节交给 AI

Vibe Coding 工作流程

提示你需要学什么？

在 AI 时代，你需要学习的不是”如何写代码”，而是： - 如何清晰地描述问题 - 如何阅读和理解代码 - 如何验证代码是否正确 - 如何发现和修复错误

1.3 Cursor AI 安装与配置

什么是 Cursor？

Cursor 是一款专为 AI 编程设计的代码编辑器，基于 VS Code 开发，内置了强大的 AI 功能：

  • Tab 补全：AI 预测你的下一步操作
  • 聊天窗口：用自然语言与 AI 对话
  • 代码生成：从描述直接生成代码
  • 代码解释：选中代码，让 AI 解释

安装步骤

  1. 下载 Cursor
    • 访问 cursor.sh
    • 下载对应操作系统版本（Windows / macOS / Linux）
    • 安装并启动
  2. 登录账号
    • 首次启动会提示登录或注册
    • 可使用 GitHub 或 Google 账号登录
    • 免费账户每月有使用额度
  3. 选择 AI 模型
    • 打开设置（Cmd+, 或 Ctrl+,）
    • 在 AI 设置中选择模型（推荐 Claude 或 GPT-4）
  4. 配置 Python 环境
    • 确认 Python 3.11+ 已安装
    • 在 Cursor 中按 Cmd+Shift+P，搜索 “Python: Select Interpreter”
    • 选择已安装的 Python 版本

Cursor 界面速览

快捷键 功能

1.4 第一次对话：Hello AI World

现在，让我们用自然语言生成第一个 Python 程序。

任务描述

我们要创建一个程序，功能是： - 输出我的名字和当前时间 - 根据时间显示不同的问候语（早上好/下午好/晚上好）

与 AI 对话

在 Cursor 中按 Cmd+L 打开聊天窗口，输入以下 Prompt：

写一个 Python 程序：
- 输出我的名字"张三"和当前时间
- 根据当前小时显示问候语：
  - 6:00-12:00 显示"早上好"
  - 12:00-18:00 显示"下午好"
  - 其他时间显示"晚上好"

AI 生成的代码

AI 可能会生成类似这样的代码：

from datetime import datetime

def get_greeting():
    """根据当前时间返回问候语"""
    hour = datetime.now().hour
    if 6 <= hour < 12:
        return "早上好"
    elif 12 <= hour < 18:
        return "下午好"
    else:
        return "晚上好"

# 主程序
name = "张三"
current_time = datetime.now().strftime("%Y-%m-%d %H:%M:%S")

print(f"{get_greeting()}，{name}！")
print(f"当前时间：{current_time}")

观察和思考

运行这段代码后，思考以下问题：

  1. AI 是如何理解你的需求的？
    • 它正确理解了”根据时间显示问候语”
    • 它自动使用了 datetime 模块
  2. 代码结构是怎样的？
    • 有一个 get_greeting() 函数
    • 有变量 name 和 current_time
    • 使用 print() 输出结果
  3. 如果想修改，该怎么做？
    • 想把 6:00 改成 5:00？直接告诉 AI
    • 想添加”中午好”？描述清楚规则
提示尝试修改

试着对 AI 说： - “把名字改成我的真名” - “添加一个’中午好’的问候语，12:00-14:00 显示” - “用英文显示问候语”

观察 AI 如何修改代码。
