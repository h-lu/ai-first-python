# 第1周：开学导论与工具配置

> **课时**：2课时 | **阶段**：环境搭建与计算思维

## 🎯 学习目标

- 理解 AI 时代编程学习的范式转变
- 完成 Cursor AI 开发环境配置
- 体验用自然语言生成第一个程序

---

## 📚 课程内容

### 1. AI 编程革命（30分钟）

**核心观点**：从"写代码"到"引导代码"

| 传统编程 | Vibe Coding |
|---------|-------------|
| 记忆语法 | 描述需求 |
| 手写代码 | AI 生成 + 人工审查 |
| 查文档 | 对话式学习 |
| 调试报错 | 让 AI 解释错误 |

**讨论话题**：
- 你觉得 AI 会取代程序员吗？
- 程序员在 AI 时代的核心价值是什么？

### 2. Andrej Karpathy 的 Vibe Coding 哲学（20分钟）

**关键概念**：
- "忘记代码的存在"——专注于描述问题
- 多层次 AI 工具：Tab Complete（75%）+ Side Chat + 高级模型
- "代码是短暂的"——快速生成、快速丢弃

**推荐观看**：Karpathy 关于 AI Coding 的访谈（可课后观看）

### 3. Cursor AI 安装与配置（40分钟）

**实操步骤**：

1. **下载安装**
   - 访问 https://cursor.sh 下载
   - 安装并启动

2. **基础配置**
   - 登录/注册账号
   - 选择 AI 模型（推荐 Claude 或 GPT-4）
   - 熟悉界面：编辑区、聊天区、终端

3. **Python 环境**
   - 确认 Python 3.11+ 已安装
   - 在 Cursor 中选择 Python 解释器

### 4. 第一次对话：Hello AI World（30分钟）

**课堂实践**：

用自然语言让 AI 生成第一个程序：

```
Prompt 示例：
"写一个 Python 程序，输出我的名字和当前时间，
并根据时间显示不同的问候语（早上好/下午好/晚上好）"
```

**观察要点**：
- AI 是如何理解你的需求的？
- 生成的代码你能看懂吗？
- 如果想修改，该如何描述？

## 🔧 课堂演示代码参考

```python
# AI 可能生成的 Hello World 示例
from datetime import datetime

def get_greeting():
    hour = datetime.now().hour
    if hour < 12:
        return "早上好"
    elif hour < 18:
        return "下午好"
    else:
        return "晚上好"

name = "张三"
print(f"{get_greeting()}，{name}！")
print(f"当前时间：{datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
```
